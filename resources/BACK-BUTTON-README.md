# Back button handler για 22450.gr

## Συμπεριφορά

- **Scroll κάτω**: Πάτα Back → scroll προς τα πάνω (μένει στη σελίδα)
- **Στην κορυφή**: Πάτα Back → προηγούμενη σελίδα (ή έξοδος αν δεν υπάρχει)

## Προσθήκη στο 22450.gr

Πρόσθεσε το script σε **όλες τις σελίδες** (ή όπου χρειάζεται):

```html
<script src="/path/to/22450-back-handler.js"></script>
```

Αντιγράψε το `22450-back-handler.js` από το `resources/` στο site του 22450.gr.

## Αλλαγή στην εφαρμογή (gate)

Το gate χρησιμοποιεί πλέον `location.replace()` αντί για `href`, ώστε η gate σελίδα να μην μένει στο history. Έτσι το Back πηγαίνει απευθείας στην προηγούμενη σελίδα του 22450.gr ή έξοδο.
